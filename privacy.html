Privacy Policy

Zo2y collects account and usage data to operate lists, profile features, collaboration, and reliability monitoring.

Data We Collect

  • Account data: email, profile metadata, and authentication state.
  • Content data: lists, ratings, notes, and collaborator links.
  • Operational telemetry: page performance, error diagnostics, and API request metadata.

How We Use Data

  • Deliver core product features and personalization.
  • Protect platform security (abuse and fraud prevention).
  • Measure product health and improve performance.

Sharing

We do not sell personal data. Data is processed by infrastructure providers required to run the service.

Retention

We retain data as needed for service delivery, legal obligations, and abuse prevention. You can request deletion via Support.

Your Choices

  • Manage analytics consent from the in-app consent banner.
  • Update account profile fields and password in settings.
  • Request account deletion via Contact.

Contact

Questions about privacy can be submitted through Contact.
